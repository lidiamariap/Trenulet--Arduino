# Trenulet--Arduino
Simulare tren cu bariera automata
## Trenulet Arduino cu bariere

## Descriere
Acest proiect simuleaza un sistem feroviar cu bariere automate realizat in Tinkercad.

## Functionalitati
- detectarea trenului
- coborarea barierei
- control cu pwm 

## Componente
- Arduino Uno
- servo motor
- senzor ultrasonic
